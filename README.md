## Постановка задачи

Необходимо разработать лексический анализатор, который выполняет разбор исходного текста программы.

Лексический анализатор должен:

-принимать на вход текст программы;
-выделять лексемы;
-классифицировать их по типам;
-определять местоположение каждой лексемы (номер строки и позиции символов);
-обнаруживать недопустимые символы и выводить сообщение об ошибке;
-поддерживать многострочный ввод.

Результат анализа должен выводиться в таблицу со следующими столбцами:

1.условный код лексемы;
2.тип лексемы;
3.лексема;
4.местоположение.

Лексический анализатор должен быть встроен в ранее разработанный графический интерфейс программы.

## Вариант задания

Лексический анализатор должен распознавать конструкцию объявления ассоциативного массива с инициализацией на языке C#.

Пример конструкции:

Dictionary<int, string> My_dict1 = new Dictionary<int, string> {
    { 1, "one" },
    { 2, "two" },
    { 3, "three" }
};
